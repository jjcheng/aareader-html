How to save web page in AA Reader

1. Open browser on your device, go to any website, tap on the action button on the bottom toolbar

2. Tap on AA Reader icon.

3. If you don't see it, tap on the More button at the rightmost

4. Tap Save button to save to AA Reader. You can optionally edit the title and select a different thumbnail. If you would like to save the HTML source code (not neccessary most of the time), turn on Save HTML

5. Open AA Reader app, go to Saves tab, the saved article is right there
